# 2 运行
## 2.1 打包方式运行
## win
直接打开release下的exe即可。

## linux
使用release下的jar运行：
```java
java -jar Calculator.jar
```

## 2.2 源码编译运行

- JDK11+
```java
java Calculator.java
```
- JDK11以下
```java
javac Calculator.java
java Calculator
```
